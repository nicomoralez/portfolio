PORTFOLIO

restauración de una fotografía antigua, se realizó mediante las herramientas de corrección de photoshop tales como el tapón de clonar, los correctores puntuales y el rellenado según el contenido para las zonas mas dañadas que tenian la misma textura. Luego de la restauración se utilizo una inteligencia artificial para poder darle un color mas fiable a la foto original.

esta es una portada creada para un fotofestival, se utilizó una tipografia con mucho peso para poder crear una máscara de recorte con la fotografía inicial.
